puts

Writes a string to stdout.

int puts (char * string);

Required Header
<stdio.h>

Return Value

This function returns zero on success, EOF indicates an error.

Parameters

string

  Output string

Remarks

The puts function writes string to stdout, the null-terminator is not copied to stdout.
