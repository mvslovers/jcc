_swab

Copies and swaps bytes in a buffer.

void _swab (char * source, char * dest, int count);

void swab (char * source, char * dest, int count);

Required Header
<string.h>

Return Value

None.

Parameters

source

  The source buffer

dest

  The destination buffer

count

  The number of bytes in the buffers

Remarks

The _swab function copies bytes from source to dest swapping even-odd pairs for transfer to machines with a different byte order.
